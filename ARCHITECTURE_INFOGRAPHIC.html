Key Design Decisions
No Redux / Zustand
One data source, one owner. Props suffice. Global state would be over-engineering.
Optimistic drag updates
State updates instantly before Supabase confirms — drag feels snappy, no snap-back.
Realtime re-fetches, not patches
Simpler and bulletproof. Avoids edge-case bugs from partial payload patching.
WIP limit: 2 layers
isDropDisabled (library) + onDragEnd guard. Defense in depth.
